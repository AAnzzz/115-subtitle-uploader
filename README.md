# 115 Subtitle Uploader v0.1

一个单用户 Web 工具：保存 115 Cookie 后，批量重命名字幕并上传到指定目录。

## v0.1 功能
- 管理员登录（支持修改密码）
- 保存并验证 115 Cookie
- 目录搜索与子目录选择
- 本地字幕批量导入（`.ass` / `.srt` / `.ssa`）
- 两种重命名方式
  - 模板重命名
  - 按目录内视频文件名匹配重命名（仅匹配视频扩展名）
- 上传前预览与人工修正
- 冲突自动重命名（如 `name (1).ext`）
- 任务历史、失败重试、日志查看
- 暗色模式（可切换，自动记忆）

## 稳定性增强（v0.1）
- 任务队列状态实时写入 SQLite：`Pending / Processing / Failed / Completed`
- 服务重启后自动恢复未完成任务（含重试）
- SQLite 并发优化（WAL + busy timeout + 索引）

## 镜像与发布
- Docker 多阶段构建（前端/后端/上传器分层）
- 运行层仅保留生产依赖，镜像更小
- 支持 GitHub Actions 自动构建并推送到 Docker Hub / GHCR

## 常见问题
- 关闭网页后任务会继续吗？
  - 会。任务在后端执行，浏览器关闭不影响。
- 系统重启后任务会丢吗？
  - 不会。未完成任务会在服务恢复后自动继续。
- 如何切换主题？
  - 页面右上角可切换浅色/暗色模式。
- 如何更新 Docker 版本？
  - 拉取新镜像后重建容器即可（例如 Dockge 里 Pull + Recreate）。
